# LAN File Share

A lightweight LAN file/text sharing tool. No installation required - just download and run.

## Features

- **File Transfer** - Drag and drop upload, configurable size limit (default 500MB)
- **Text Sharing** - One-click copy/paste text
- **LAN Access** - Access from phones, computers, and other devices
- **IE Compatible** - Compatible with IE8 and above
- **Settings** - Configurable upload size limit, one-click service stop
- **Data Storage** - All data stored locally for privacy

## Usage

1. Double-click `LanFileShare.exe`
2. Wait a few seconds for browser to open, or manually visit the displayed address
3. Other devices on the LAN can access via `http://[IP_ADDRESS]:5000`

## Stop Service

Click the "Stop Service" button at the bottom of the page

## Data Directories

The following directories are created automatically:

- `uploads/` - Uploaded file storage
- `text_data.json` - Saved text data
- `config.json` - Configuration file

## System Requirements

- Windows 7 or above (64-bit)
- No additional runtime required
