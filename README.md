# 局域网文件中转站

一款轻量级的局域网文件/文本分享工具，无需安装任何环境，下载即用。

## 功能特性

- 📁 **文件传输** - 支持拖拽上传，最大可配置（默认500MB）
- 📝 **文本分享** - 一键复制/粘贴文本
- 🌐 **局域网访问** - 支持手机、电脑等多设备访问
- 🌐 **IE兼容** - 兼容IE8及以上浏览器
- ⚙️ **设置功能** - 可配置上传大小限制，一键停止服务
- 💾 **数据存储** - 所有数据存储在本地，保护隐私

## 使用方法

1. 下载并解压 `LanFileShare_Portable` 文件夹
2. 双击运行 `LanFileShare.exe`
3. 等待几秒会自动打开浏览器，或手动访问显示的地址
4. 局域网其他设备通过 `http://[IP地址]:5000` 访问

## 停止服务

- 方法1：点击页面底部的"停止服务"按钮
- 方法2：直接关闭命令行窗口

## 数据目录

运行后会自动创建以下目录：
- `uploads/` - 上传的文件存储目录
- `text_data.json` - 保存的文本数据
- `config.json` - 配置文件

## 系统要求

- Windows 7 及以上（64位）
- 无需安装任何运行环境
